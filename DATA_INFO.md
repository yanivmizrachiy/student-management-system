# ⚠️ מידע חשוב על נתוני התלמידים

## הבעיה:

כשאתה פותח את האתר דרך **GitHub Pages** (הקישור הקבוע), הנתונים שאתה רואה הם **לא** הנתונים שלך!

### למה זה קורה?

הנתונים נשמרים ב-**localStorage** של הדפדפן. זה אומר:
- כל דפדפן יש לו localStorage משלו
- כשפותחים את האתר דרך GitHub Pages, זה נחשב ל"דפדפן חדש"
- ה-localStorage ריק, אז אין נתונים

## פתרונות:

### פתרון 1: פתיחת האתר מקומית (מומלץ)
1. פתח את הקובץ `index.html` **ישירות מהמחשב** (לא דרך GitHub Pages)
2. דרך 1: לחץ כפול על `index.html` בתיקיית הפרויקט
3. דרך 2: גרור את `index.html` לדפדפן
4. הנתונים המקומיים שלך יופיעו!

### פתרון 2: ייבוא נתונים מחדש
אם אתה רוצה להשתמש ב-GitHub Pages:
1. פתח את `init_data.html` **מקומית**
2. לחץ על הכפתורים כדי לייבא את הנתונים
3. הנתונים יישמרו ב-localStorage של הדפדפן הזה
4. עכשיו כשתפתח את GitHub Pages באותו דפדפן - תראה את הנתונים

### פתרון 3: שמירת נתונים ב-GitHub (עתידי)
אפשר לעדכן את המערכת לשמור נתונים ב-GitHub במקום localStorage, אבל זה דורש שינויים משמעותיים.

## מה לעשות עכשיו?

**הכי פשוט:** פתח את `index.html` **מקומית** - זה יעבוד עם הנתונים שלך!
