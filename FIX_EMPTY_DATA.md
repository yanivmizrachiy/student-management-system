# ⚠️ בעיה: 0 תלמידים במערכת - איך לפתור

## מה הבעיה?

הנתונים נשמרים ב-**localStorage** של הדפדפן. זה אומר:
- כשפותחים דרך **GitHub Pages** → localStorage ריק (דומיין אחר)
- כשפותחים **מקומית** → localStorage המקומי (עם הנתונים שלך)

## פתרונות:

### פתרון 1: פתיחת האתר מקומית (הכי מהיר!)

1. פתח את התיקייה: `C:\Users\yaniv\OneDrive\שולחן העבודה\Glide לניהול תלמידים_files`
2. לחץ **כפול** על `index.html`
3. האתר ייפתח עם הנתונים המקומיים שלך!

### פתרון 2: ייבוא נתונים דרך init_data.html

אם אתה רוצה להשתמש ב-GitHub Pages:

1. פתח את `init_data.html` **מקומית** (כפול עליו)
2. לחץ על הכפתורים כדי לייבא את כל הקבוצות
3. הנתונים יישמרו ב-localStorage של הדפדפן הזה
4. עכשיו כשתפתח את GitHub Pages **באותו דפדפן** → תראה את הנתונים

### פתרון 3: העתקת localStorage

1. פתח את האתר **מקומית** (`index.html`)
2. לחץ F12 (כלי מפתחים)
3. Console → כתוב:
   ```javascript
   localStorage.getItem('schoolData')
   ```
4. העתק את התוצאה
5. פתח את GitHub Pages
6. F12 → Console → כתוב:
   ```javascript
   localStorage.setItem('schoolData', 'הדבק כאן את מה שהעתקת')
   ```
7. רענן את הדף (F5)

## המלצה:

**הכי פשוט:** פתח את `index.html` מקומית - זה יעבוד עם הנתונים שלך מיד!
