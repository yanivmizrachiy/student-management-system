# 🌐 הגדרת GitHub Pages - קישור קבוע לאתר

## ✅ מה זה?

GitHub Pages מאפשר לך לפרסם את האתר שלך עם קישור קבוע וציבורי:
**https://yanivmizrachiy.github.io/student-management-system/**

הקישור הזה יהיה זמין תמיד, גם אחרי עדכונים!

## 🚀 איך להפעיל:

### אופציה 1: סקריפט אוטומטי (מומלץ)
```powershell
.\setup-github-pages.ps1
```

זה יעשה הכל אוטומטית:
- ✅ יגדיר GitHub Pages
- ✅ יוצר קיצור דרך בשולחן העבודה
- ✅ פותח את האתר בדפדפן

### אופציה 2: ידנית דרך GitHub

1. היכנס ל-GitHub: https://github.com/yanivmizrachiy/student-management-system
2. לחץ על **Settings** (הגדרות)
3. בתפריט הצד, לחץ על **Pages**
4. תחת **Source**, בחר:
   - Branch: **main**
   - Folder: **/ (root)**
5. לחץ **Save**
6. המתן 1-2 דקות
7. האתר יהיה זמין ב: **https://yanivmizrachiy.github.io/student-management-system/**

## 📁 קיצור דרך בשולחן העבודה:

### יצירה אוטומטית:
```powershell
.\create-desktop-shortcut.ps1
```

### יצירה ידנית:

1. לחץ ימני על שולחן העבודה
2. בחר **New** → **Shortcut**
3. הזן את הכתובת:
   ```
   https://yanivmizrachiy.github.io/student-management-system/
   ```
4. לחץ **Next**
5. תן שם: **מערכת ניהול תלמידים**
6. לחץ **Finish**

## ⚙️ עדכונים:

כל עדכון שתדחוף ל-GitHub יעודכן אוטומטית באתר תוך 1-2 דקות!

```bash
git add .
git commit -m "עדכון"
git push origin main
```

## 📋 הערות חשובות:

1. **זמן פעילות:** GitHub Pages יכול לקחת 1-2 דקות להיכנס לפעילות בפעם הראשונה
2. **Repository חייב להיות Public:** אם ה-repository שלך private, תצטרך לשדרג ל-GitHub Pro או להפוך אותו ל-public
3. **קובץ index.html:** הקבץ `index.html` חייב להיות בתיקיית ה-root של ה-repository (כפי שכבר יש לנו)

2. ✅ GitHub Pages מוגדר (או יוגדר על ידי הסקריפט)
3. ✅ קיצור דרך בשולחן העבודה נוצר
4. ✅ האתר זמין תמיד בכתובת הקבועה
