# 🌐 תצוגה חיה של האתר - מדריך שימוש

## 🚀 הפעלה מהירה

### שיטה 1: הפעלה אוטומטית (מומלץ)
לחץ פעמיים על אחד מהקבצים:
- **`start-live-view.bat`** (Windows)
- **`start-live-view.ps1`** (PowerShell)

זה יפתח אוטומטית:
- ✅ שרת על פורט 8000
- ✅ דפדפן עם האתר
- ✅ Keep-Alive שומר שהשרת תמיד רץ

### שיטה 2: הפעלה ידנית

1. **הפעל את השרת:**
   ```bash
   python server.py
   ```

2. **פתח דפדפן:**
   - גש ל: `http://localhost:8000/index.html`

3. **הפעל Keep-Alive (אופציונלי):**
   ```bash
   python keep-server-alive.py
   ```

## 🔄 איך זה עובד?

### רענון אוטומטי
- השרת בודק כל **שנייה** אם יש שינויים בקבצים
- הדפדפן בודק כל **2 שניות** אם יש עדכונים
- כשמזוהה עדכון, הדף מתעדכן **אוטומטית**

### Keep-Alive
- בודק כל **5 שניות** אם השרת רץ
- אם השרת נפל, מפעיל אותו **אוטומטית**
- פותח דפדפן אם הוא לא פתוח

### אינדיקטור ויזואלי
בפינה הימנית העליונה תראה:
- 🟢 **ירוק** - רענון אוטומטי פעיל
- 🟡 **צהוב** - מנסה להתחבר
- 🟠 **כתום** - מרענן...
- 🔴 **אדום** - חיבור נכשל

## 📁 קבצים חשובים

- **`server.py`** - השרת הראשי
- **`keep-server-alive.py`** - שומר שהשרת תמיד רץ
- **`start-live-view.ps1`** - סקריפט הפעלה (PowerShell)
- **`start-live-view.bat`** - סקריפט הפעלה (Batch)
- **`live-reload.js`** - רענון אוטומטי בדפדפן

## ⚙️ הגדרות

### שינוי פורט
אם פורט 8000 תפוס, שנה ב-`server.py`:
```python
PORT = 8000  # שנה למספר אחר
```

### התקנת תלויות
אם Keep-Alive לא עובד, התקן:
```bash
pip install psutil
```

## 🛑 עצירה

לעצור את השרת:
- לחץ `Ctrl+C` בחלון השרת
- או סגור את חלון השרת

## 💡 טיפים

1. **תמיד השתמש ב-`start-live-view.bat`** - זה הכי נוח
2. **שמור את החלון פתוח** - Keep-Alive יעבוד רק אם הסקריפט רץ
3. **אם הדף לא מתעדכן** - רענן ידנית (F5) או בדוק את הקונסול
4. **אם השרת לא מתחיל** - בדוק שהפורט פנוי או שנה פורט

## 🔧 פתרון בעיות

### השרת לא מתחיל
```
❌ Port 8000 is already in use
```
**פתרון:** סגור תהליכים ישנים או שנה פורט

### הדף לא מתעדכן
1. בדוק את הקונסול בדפדפן (F12)
2. בדוק שהשרת רץ
3. רענן ידנית (F5)

### Keep-Alive לא עובד
1. התקן: `pip install psutil`
2. הפעל ידנית: `python keep-server-alive.py`
