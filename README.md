# מערכת ניהול תלמידים

מערכת מקצועית ויוקרתית לניהול תלמידים בבית ספר.

## תכונות עיקריות

- ✅ ניהול תלמידים, כיתות, הקבצות ומורים
- ✅ מערכת ציונים דינמית (5+ עמודות)
- ✅ העלאת תמונות וקבצים לכל תלמיד
- ✅ מערכת הרשאות (בעל מערכת / צופה)
- ✅ עיצוב פרימיום יוקרתי
- ✅ מותאם למובייל
- ✅ שמירה אוטומטית ב-localStorage

## שימוש

1. פתח את `index.html` בדפדפן
2. השתמש ב-`init_data.html` להזנת נתונים ראשונית
3. המערכת שומרת אוטומטית ב-localStorage

## קבצים עיקריים

- `index.html` - דף הבית
- `layer.html` - עמוד שכבה
- `group.html` - עמוד הקבצה
- `student.html` - עמוד תלמיד
- `teacher.html` - עמוד מורה
- `class.html` - עמוד כיתה
- `data.js` - מאגר נתונים מרכזי
- `init_data.html` - הזנת נתונים ראשונית

## הרשאות

למעבר למצב בעל מערכת - לחץ על כפתור ההרשאות בפינה השמאלית העליונה.

## טכנולוגיות

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage
