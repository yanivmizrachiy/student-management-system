# 🚀 הפעלת סנכרון אוטומטי ל-GitHub

## ✅ מה זה עושה?
כל עדכון שאתה מבצע באתר (הוספת תלמיד, שינוי ציון, העלאת קובץ וכו') **יועלה אוטומטית ל-GitHub תוך מספר שניות!**

## 🎯 איך להפעיל:

### אופציה 1: קובץ Batch (הכי פשוט)
**כפול על:** `start-auto-sync.bat`
- זה יבחר אוטומטית בין Node.js ל-PowerShell
- הרץ פעם אחת ותן לו לרוץ ברקע

### אופציה 2: Node.js (מומלץ)
```bash
npm start
```
או:
```bash
node auto-sync-watcher.js
```

### אופציה 3: PowerShell
```powershell
.\auto-sync-watcher.ps1
```

## ⚙️ איך זה עובד?
1. אתה מבצע שינוי באתר (למשל: מעדכן ציון)
2. המערכת שומרת ב-localStorage
3. נוצר קובץ marker (`.sync-marker.json`)
4. ה-watcher מזהה את הקובץ תוך 3 שניות
5. ממתין 5 שניות לאיסוף שינויים מרובים
6. מבצע `git add`, `commit`, `push` אוטומטית
7. הכל מופיע ב-GitHub! 🎉

## 📋 דרישות:
- ✅ Git מותקן ומוגדר
- ✅ GitHub CLI מותקן (gh) - כבר מותקן אצלך!
- ✅ Node.js (אופציונלי - אם רוצה את ה-JavaScript watcher)
- ✅ PowerShell (מותקן כבר ב-Windows)

## ⚠️ הערות:
- ה-watcher צריך לרוץ ברקע כל הזמן
- אם תסגור אותו, הסנכרון יפסיק
- שינויים יאספו ויעלו יחד כל 5 שניות (לא כל שינוי בנפרד)
- אין צורך לסגור את הדפדפן
