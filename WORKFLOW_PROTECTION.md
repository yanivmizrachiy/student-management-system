# מנגנון הגנה מפני הרס קוד - Workflow Protection

## מטרה
למנוע שבירת קוד קיים בעת ביצוע שינויים.

## איך זה עובד

### שלב 1: לפני כל עריכה חשובה

**לפני ביצוע שינוי בקובץ, יש לבדוק:**

1. **קריאת הקובץ המלא:**
   ```bash
   # קרא את הקובץ המלא כדי להבין את המבנה
   cat layer.html
   ```

2. **זיהוי אלמנטים קריטיים:**
   - אילו אלמנטים DOM חייבים להישאר?
   - אילו פונקציות JavaScript חייבות להישאר?
   - אילו Scripts חייבים להיטען?
   - אילו Classes CSS חייבות להישאר?

3. **יצירת רשימת בדיקות:**
   - רשום את כל האלמנטים הקריטיים
   - ודא שאתה יודע מה התפקיד של כל אחד

### שלב 2: בדיקה לפני עריכה

**בדוק שהקובץ תקין לפני שינוי:**

```javascript
// בדוק שהקובץ תקין
const checks = [
  'id="layerTitle" קיים',
  'id="groupsGrid" קיים',
  'function renderPage קיימת',
  'src="data.js" קיים',
  'src="shared-utils.js" קיים',
  'src="live-reload.js" קיים'
];
```

### שלב 3: ביצוע השינוי

**בצע את השינוי תוך שמירה על:**
- ✅ כל האלמנטים הקריטיים
- ✅ כל הפונקציות הקריטיות
- ✅ כל ה-Scripts
- ✅ כל ה-Classes CSS

### שלב 4: בדיקה אחרי עריכה

**אחרי השינוי, בדוק שוב:**

1. **אימות שהכל עדיין קיים:**
   ```bash
   # בדוק שהאלמנטים עדיין קיימים
   grep -c 'id="layerTitle"' layer.html
   grep -c 'id="groupsGrid"' layer.html
   grep -c 'function renderPage' layer.html
   ```

2. **בדיקת שגיאות:**
   ```bash
   # אם יש linter, הרץ אותו
   # או בדוק ידנית
   ```

3. **בדיקה בדפדפן:**
   - פתח את הדף בדפדפן
   - בדוק שהכל עובד
   - בדוק את הקונסול לשגיאות

## רשימת בדיקות ל-layer.html

### אלמנטים DOM שצריכים להיות קיימים:
- ✅ `id="layerTitle"` - כותרת השכבה
- ✅ `id="groupsGrid"` - Grid של הקבוצות
- ✅ `id="header"` - Header של העמוד
- ✅ `.nav-buttons` - כפתורי ניווט
- ✅ `.group-card` - כרטיסי קבוצות

### פונקציות JavaScript שצריכות להיות קיימות:
- ✅ `getLayerFromURL()` - קבלת שכבה מ-URL
- ✅ `renderPage()` - רינדור העמוד
- ✅ `init()` - אתחול

### Scripts שצריכים להיטען:
- ✅ `data.js` - מאגר נתונים
- ✅ `shared-utils.js` - כלי עזר משותפים
- ✅ `auto-load-data.js` - טעינת נתונים אוטומטית
- ✅ `live-reload.js` - רענון אוטומטי

### Classes CSS שצריכות להיות קיימות:
- ✅ `.group-card` - כרטיס קבוצה
- ✅ `.group-title` - שם הקבוצה
- ✅ `.group-teacher` - שם המורה
- ✅ `.group-count-badge` - מונה תלמידים
- ✅ `.nav-btn` - כפתור ניווט

### מבנה HTML בסיסי:
- ✅ `<div class="container">`
- ✅ `<div class="header" id="header">`
- ✅ `<h1 id="layerTitle">`
- ✅ `<div class="groups-grid" id="groupsGrid">`
