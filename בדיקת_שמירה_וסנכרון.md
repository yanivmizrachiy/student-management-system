# בדיקת שמירה וסנכרון - מערכת ניהול תלמידים

## ✅ מערכת השמירה

### 1. מאגר נתונים מרכזי (data.js)
- ✅ כל הנתונים נשמרים ב-`localStorage` תחת המפתח `'schoolData'`
- ✅ כל ישות נשמרת: `students`, `classes`, `groups`, `teachers`, `gradeColumns`
- ✅ שמירה אוטומטית בכל פעולה: `DataStore.save()`

### 2. פונקציות שמירה וטעינה

#### `load()` - טעינת נתונים
```javascript
- קורא מ-localStorage את 'schoolData'
- מפרק JSON
- ממלא את כל המערכים
- יוצר 5 עמודות ציונים ברירת מחדל אם אין
```

#### `save()` - שמירת נתונים
```javascript
- יוצר אובייקט עם כל הנתונים
- ממיר ל-JSON
- שומר ב-localStorage
- מחזיר true (הצלחה)
```

### 3. נקודות שמירה במערכת

#### אוטומטיות:
- ✅ אחרי `importGroup()` - הוספת קבוצה
- ✅ אחרי `saveGrade()` - עדכון ציון
- ✅ אחרי `uploadImage()` - העלאת תמונה
- ✅ אחרי `uploadFiles()` - העלאת קבצים
- ✅ אחרי `deleteFile()` - מחיקת קובץ
- ✅ אחרי `updateStudentField()` - עדכון שם/הערות

#### ידניות:
- ✅ `clearAll()` - איפוס כל הנתונים

### 4. סנכרון אוטומטי

#### בכל עמוד:
- ✅ `DataStore.load()` נקרא ב-DOMContentLoaded
- ✅ עדכון אוטומטי כל 2 שניות (`setInterval`)
- ✅ מונים מתעדכנים אוטומטית

#### במונים:
- ✅ כל פונקציית מונה קוראת ל-`load()` לפני החישוב
- ✅ מבטיח נתונים עדכניים תמיד

## 🔍 בדיקת תקינות

### איך לבדוק שהכל נשמר:

1. **פתח את Console (F12)**
2. **הרץ:**
```javascript
// בדיקת שמירה
DataStore.load();
console.log('תלמידים:', DataStore.students.length);
console.log('כיתות:', DataStore.classes.length);
console.log('הקבצות:', DataStore.groups.length);
console.log('מורים:', DataStore.teachers.length);

// בדיקת localStorage ישירות
const saved = localStorage.getItem('schoolData');
console.log('נתונים שמורים:', saved ? JSON.parse(saved) : 'אין נתונים');
```

3. **בדוק עדכון:**
```javascript
// שנה משהו במערכת
// ואז רץ שוב:
DataStore.load();
console.log('נתונים מעודכנים:', DataStore.students.length);
```

### בדיקת סנכרון בין טאבים:

1. פתח את המערכת בשני טאבים
2. שנה משהו בטאב אחד
3. חכה 2 שניות
4. רענן את הטאב השני (F5)
5. הנתונים אמורים להיות זהים

## ⚠️ הערות חשובות:

1. **localStorage הוא מקומי** - הנתונים נשמרים רק בדפדפן הספציפי
2. **אין סנכרון אוטומטי בין מכשירים** - כל דפדפן נשמר בנפרד
3. **אם רוצים סנכרון עם GitHub** - צריך להוסיף:
   - Git repository
   - GitHub Actions או script לסנכרון
   - API integration
