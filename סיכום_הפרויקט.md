# סיכום הפרויקט - מערכת ניהול תלמידים

## סקירה כללית
נוצרה מערכת ניהול תלמידים מלאה ומתקדמת עם ממשק משתמש מודרני ונוח. המערכת כוללת ניהול תלמידים, כיתות, הקבצות, מורים וציונים.

---

## קבצים שנוצרו

### 1. **index.html** - דף הבית
- דף ראשי עם ספירת תלמידים לפי שכבות (ז', ח', ט')
- כפתורים מעוצבים לכל שכבה עם צבעים שונים
- מונה כולל של כל התלמידים
- מערכת הרשאות (מצב צפייה / מצב בעל מערכת)
- עיצוב מודרני עם רקע כהה וגרדיאנטים

### 2. **data.js** - מאגר נתונים מרכזי
- **DataStore** - אובייקט מרכזי לניהול כל הנתונים
- שמירה וטעינה מ-localStorage
- פונקציות עזר לספירה וסינון:
  - `getLayerCount()` - ספירת תלמידים לפי שכבה
  - `getGroupCount()` - ספירת תלמידים לפי הקבצה
  - `getClassCount()` - ספירת תלמידים לפי כיתה
  - `getTotalCount()` - ספירה כוללת
- מבנה נתונים:
  - `students` - רשימת תלמידים
  - `classes` - רשימת כיתות
  - `groups` - רשימת הקבצות
  - `teachers` - רשימת מורים
  - `gradeColumns` - עמודות ציונים דינמיות

### 3. **layer.html** - דף שכבה
- הצגת כל ההקבצות בשכבה מסוימת
- כרטיסים מעוצבים לכל הקבצה
- מידע על מורה ממונה לכל הקבצה
- ספירת תלמידים בכל הקבצה
- ניווט חזרה לדף הבית

### 4. **group.html** - דף הקבצה
- טבלת תלמידים מפורטת עם:
  - שם פרטי ושם משפחה
  - כיתה
  - מחנכת
  - עמודות ציונים דינמיות
- אפשרות להוספת עמודות ציונים (לבעלי הרשאות)
- עריכת ציונים ישירות מהטבלה
- ניווט דרך breadcrumbs

### 5. **student.html** - דף תלמיד
- כרטיס תלמיד מפורט עם:
  - תמונה (ניתן להעלות)
  - פרטים אישיים (שם, שכבה, הקבצה, כיתה, מחנכת)
  - ציונים לפי עמודות
  - קבצים מצורפים (ניתן להעלות)
- אפשרות להעלאת תמונות וקבצים (לבעלי הרשאות)
- עריכת ציונים ישירה

### 6. **class.html** - דף כיתה
- טבלת תלמידי הכיתה
- מידע על מחנכת הכיתה
- קישורים לפרטי כל תלמיד
- עיצוב נקי ופשוט

### 7. **teacher.html** - דף מורה
- הצגת כל ההקבצות שהמורה ממונה עליהן
- כרטיסים מעוצבים לכל הקבצה
- מידע על שכבה ומספר תלמידים

---

## תכונות עיקריות

### 1. **מערכת הרשאות**
- מצב צפייה - רק צפייה בנתונים
- מצב בעל מערכת - עריכה, הוספה והעלאת קבצים
- החלפה בין מצבים בלחיצת כפתור

### 2. **ניהול ציונים**
- עמודות ציונים דינמיות (ניתן להוסיף)
- עריכת ציונים ישירה מהטבלה או מדף התלמיד
- שמירה אוטומטית ב-localStorage

### 3. **ניהול קבצים**
- העלאת תמונות לתלמידים
- העלאת קבצים מצורפים
- שמירה ב-localStorage כ-base64

### 4. **ניווט אינטואיטיבי**
- Breadcrumbs בכל דף
- קישורים בין כל הרכיבים (תלמיד → כיתה → מורה → הקבצה)
- כפתורי חזרה וניווט

### 5. **עיצוב מודרני**
- רקע כהה עם גרדיאנטים
- אפקטי hover ואנימציות
- עיצוב responsive (מותאם למובייל)
- צבעים שונים לכל שכבה:
  - שכבה ז' - כחול (#2962ff)
  - שכבה ח' - ירוק (#00c853)
  - שכבה ט' - כתום (#ff9800)

### 6. **עדכון אוטומטי**
- רענון נתונים כל 2 שניות
- סנכרון בין כל הדפים

```javascript
{
  id: string,
  firstName: string,
  lastName: string,
  layer: '7' | '8' | '9',
  groupId: string,
  classId: string,
  image: string (base64),
  files: [{id, name, url}],
  grades: {columnId: grade}
}
```

```javascript
  name: string,
  layer: '7' | '8' | '9',

```

## שימוש במערכת

### כבעל מערכת:
1. לחץ על כפתור "🔓 מצב צפייה" כדי להחליף ל"🔒 מצב בעל מערכת"
2. כעת תוכל:
   - להוסיף עמודות ציונים
   - לערוך ציונים
   - להעלות תמונות וקבצים

- רק צפייה בנתונים ללא אפשרות עריכה

- תמונות וקבצים נשמרים כ-base64 (יש מגבלת גודל)
